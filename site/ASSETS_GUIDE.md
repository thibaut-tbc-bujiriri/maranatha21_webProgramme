# Guide des Assets

## Hiérarchie
- assets/images: images génériques et logos
- assets/images/Album: photos de la galerie
- assets/images/videos: vidéos (invitation)

## Conventions
- Noms en minuscules avec underscores
- Formats: jpg/png pour photos, mp4 pour vidéos
- Pas d’espaces ni accents dans les chemins

## Bonnes pratiques
- Utiliser loading=\"lazy\" sur les images
- Précharger les visuels critiques si nécessaire
- Centraliser les listes dans js/assets.js
