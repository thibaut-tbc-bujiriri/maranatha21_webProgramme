# Images du Site Maranatha 21

Ce dossier doit contenir les images suivantes :

## Images Requises

### favicon.ico
- **Taille recommandée** : 16x16, 32x32 ou 48x48 pixels
- **Format** : .ico ou .png
- **Description** : Icône du site affichée dans l'onglet du navigateur
- **Suggestion** : Logo ou symbole représentant Maranatha 21

### apotre-placeholder.jpg
- **Taille recommandée** : 400x400 pixels minimum (carré)
- **Format** : .jpg, .png ou .webp
- **Description** : Photo de l'apôtre de l'église
- **Usage** : Page "À propos" (about.html)

### speaker1-placeholder.jpg
- **Taille recommandée** : 400x400 pixels minimum (carré, sera affiché en cercle)
- **Format** : .jpg, .png ou .webp
- **Description** : Photo du premier intervenant (Pasteur Jean-Baptiste Mwamba)
- **Usage** : Page "Intervenants" (speakers.html)

### speaker2-placeholder.jpg

## Notes

- Toutes les images doivent être optimisées pour le web (compression appropriée)
- Les images de speakers seront affichées en cercle (border-radius: 50%)
- Utilisez des images de bonne qualité mais optimisées pour un chargement rapide
- Formats recommandés : WebP (meilleure compression) ou JPG/PNG (compatibilité)

## Sources d'Images Gratuites

Si vous avez besoin de placeholders temporaires, vous pouvez utiliser :
- [Unsplash](https://unsplash.com) - Photos gratuites
- [Pexels](https://www.pexels.com) - Photos gratuites
- [Placeholder.com](https://placeholder.com) - Images placeholder

## Remplacement

Pour remplacer les placeholders :
1. Ajoutez vos images dans ce dossier (`assets/images/`)
2. Assurez-vous que les noms de fichiers correspondent exactement aux noms utilisés dans le code HTML
3. Vérifiez que les images s'affichent correctement dans le navigateur
